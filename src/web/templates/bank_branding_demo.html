API Endpoints

The following API endpoints are available for bank branding:

  • /api/bank-branding/scotiabank - Get complete branding info
  • /api/bank-branding/scotiabank/colors - Get color scheme
  • /api/bank-branding/scotiabank/logo - Get logo information
  • /api/bank-branding/scotiabank/theme - Get theme information
  • /api/bank-branding/all - Get all bank branding
  • /api/bank-branding/available - Get available banks
